# Rock, Paper, Scissors (Python)

A simple **command-line Rock, Paper, Scissors game** built using Python.  
Play against the computer with live score tracking and the option to replay.

---

## Features

- Play Rock, Paper, Scissors against the computer  
- Tracks user and computer scores  
- Input validation for user choices  
- Option to play multiple rounds  
- Fun and interactive command-line interface  

---

## Technologies Used

- Python 3  
- `random` module for computer choices  
- Loops and conditionals
